# 香水收藏室 — 上線與使用說明

這是一個單一網頁 App。資料夾裡有四個檔案，全部一起上傳即可：

- `index.html`（主程式）
- `manifest.webmanifest`（讓它能加到手機主畫面當 App）
- `icon.svg`（App 圖示）
- `README.md`（這份說明，不影響執行，可不上傳）

---

## 最簡單：用 Netlify Drop 上線（不用指令、約 3 分鐘）

1. 電腦瀏覽器打開 https://app.netlify.com/drop
2. 把整個 `perfume-app` 資料夾直接拖進那個頁面。
3. 等幾秒，它會給你一個網址，例如 `https://隨機名稱.netlify.app`。
4. 完成。這個網址永久有效，手機電腦都能開。

> 想換成好記的網址、或日後更新內容，可以免費註冊 Netlify 帳號，把站台「claim」起來。

---

## 進階：用指令上線（你說會一點基礎指令）

需要先裝 Node.js。然後：

```bash
cd perfume-app
npx netlify-cli deploy --dir . --prod
```

第一次會要你登入並建立站台，照著提示按即可。完成後一樣給你一個網址。

GitHub Pages 也行：把這資料夾推到一個 repo，到 Settings → Pages 選分支根目錄，等它給你網址。

---

## 手機上當 App 用

1. 手機瀏覽器（iPhone 用 Safari、Android 用 Chrome）打開上一步拿到的網址。
2. iPhone：點分享鈕 → 「加入主畫面」。Android：選單 →「加到主畫面 / 安裝應用程式」。
3. 主畫面會出現圖示，點開就是全螢幕的 App。

---

## 關於資料保存（重要）

- 你新增的收藏存在「這支手機的這個瀏覽器」裡（IndexedDB），關掉、重開都還在。
- 但它是綁定瀏覽器的：換手機、換瀏覽器、或清除瀏覽器資料，收藏不會自動跟著走。
- 內建的 21 款香水庫資料是寫在程式裡的，永遠都在，不受影響。

如果你之後想要「跨裝置同步」或「雲端備份」，那需要再接一個後端資料庫，
那是另一個層級的工程，到時候再跟我說，我幫你規劃。
